Кликните на одном из мест выше, чтобы посмотреть некоторые наиболее высокие здания.
Подвигайте карту мышкой или клавишами, чтобы увидеть эффект перспективы. Нажав на иконку «прицела», вы можете увидеть своё текущее местоположение.

Движок основан на Leaflet. Геометрия зданий извлечена из файлов OpenStreetMap. В базу внесена Россия и некоторые соседние страны, итого более 7 млн объектов. Архивы по регионам доступны для скачивания.

OSM Buildings использует только операции с Canvas 2D — это не WebGL.
Суммарный размер минифицированной библиотеки — 8,6К (3,9К при сжатии gzip).
